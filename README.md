# 🏥 Hospital Management System API

A Django REST Framework-based Hospital Management System with role-based access control for managing doctors, headquarters, sub-headquarters, and visit records.

---

## 🚀 Tech Stack

- Python 3.11
- Django 5.x
- Django REST Framework
- Django Filters
- SQLite (development)

---

## ✨ Features

### 👤 Role-Based Access Control
- SUPER_ADMIN → full access
- HQ_ADMIN → manages their HQ
- HQ_STAFF → HQ-level access
- SUB_HQ_STAFF → Sub HQ-level access
- MR → Visit management

---

### 🏥 Modules

- Doctor Management
- HeadQuarter Management
- SubHeadQuarter Management
- Visit Management

---

### 🔍 API Features

- Filtering (Django Filter)
- Search functionality
- Ordering support
- Role-based queryset filtering
